## Installation

You can install this package using either `yarn` or `npm`.

Using Yarn:

```
yarn install
```

Using npm:

```
npm install
```

## Run Project

Start Server:

```
node server.js
```

Start Client:

Using Yarn:

```
yarn start
```

Using npm:

```
npm start
```
